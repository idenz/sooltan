## Requirement
- NodeJS

### Setup project
```sh
# Clone project
> git clone https://github.com/idenz/sooltan.git
> cd sooltan

# Install package
> npm install

# Change to global project
> npm install -g .
```

### Start Project

```sh
# Running tools with global project
> mytools <directory_log>

# Running tools without global project
> node denny_dharmawan.js <directory_log>
```
